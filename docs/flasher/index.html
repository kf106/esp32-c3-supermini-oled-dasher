ESP32-C3 Supermini OLED — Square Dash

Install the latest firmware in your browser. You need a USB data cable, Google Chrome or Microsoft Edge on a computer (not phone or tablet), and an ESP32-C3 Supermini OLED board.

  1. Plug the board into your computer with USB.
  2. Click Install firmware below and allow access to the serial port when asked.
  3. Pick the board’s USB serial port (often USB JTAG/serial or /dev/ttyACM0 on Ubuntu).
  4. Wait until flashing finishes. You should see the SQUARE DASH splash, then the game.
This installer needs Chrome or Edge on a desktop/laptop (Web Serial is not available here). Open this page over HTTPS (for example via GitHub Pages), not as a local file.
Tip: Hold BOOT for 2 seconds during the splash to wipe saved progress. To put the chip into download mode manually, hold BOOT while pressing RESET.
